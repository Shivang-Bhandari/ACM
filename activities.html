WORKSHOP : IOT

ACM Student Chapter (School of Computer Sciences) in association with Netmax Technologies private Ltd. organized a workshop on “Internet of Things” on 11th April,2016 where students participated with full enthusiasm and developed their technical skills. Mr. Jagdeep Singh, from Netmax technologiesPvt Ltd. provided the students with valuable information and insight on IoT. Students learned about the basics of Python and integration of electronic devices using Python

14April

WORKSHOP : PHP SCRIPTING

ACM Student Chapter(School of Computer Sciences), Chitkara University organized a Workshop on “PHP Scripting and Database Connectivity” on 14 April 2016 under the guidance of Mr. Naveen Kalra, CEO, SatShri Technologies Pvt. Ltd. Around 60 students attended the workshop with great enthusiasm and zeal. The session started with introduction to various back end languages like Ajax, MySQL and PHP..They created databases and tables using PHP and WAMP server, also connected the HTML page with database using PHP scripts.
